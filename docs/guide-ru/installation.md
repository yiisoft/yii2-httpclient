Установка
============

## Установка расширения

Для установки расширения используйте Composer. Запустите в консоли
                                            
```
composer require --prefer-dist yiisoft/yii2-httpclient "*"
```

или добавьте

```json
"yiisoft/yii2-httpclient": "*"
```
в `require` секцию вашего composer.json.
